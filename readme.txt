With this plugin it is possible to record game results and generate tables and rankings.

== Description ==
Competitions, players and the results are entered in the backend. The tables and rankings calculated from this can be displayed on the front end.
The goal of this plugin is to calculate a ranking regardless of the number of games played. For the individual ranking, the duels between the players are rated.
Points are awarded for the double ranking, which are determined by the strength of the opponents.

**FEATURES OF THE FREE VERSION**
* Configuration of a competition
* Manager players and results
* Calculate the standing and ranking based on the configuration
* Show standing, ranking and matix at the front end

**FEATURES OF THE PREMIUM VERSION**
* All of the FREE VERSION

A detailed list of all functions can be found on <a href="https://www.software-kunze.de/plugins/player-leaderboard" target="_blank">https://www.software-kunze.de/plugins/player-leaderboard</a>.
